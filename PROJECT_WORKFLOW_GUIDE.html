Project MEEEM Marketplace Quick User Guide

Easy Guide: Pick Your Panel

This page explains how the system works for non-technical users. Use it like a checklist: choose your role/panel on the left, then follow only the steps inside that panel.

Panels

Shopping (Public Customer View)

Start shopping (Public)
1 Open the Home page
`/`
Home shows categories and an “Explore products” grid. For now, the “Add to Cart” button appears only on Explore.
2 Browse all items
`/browse`
Browse supports category/subcategory filtering and shows product cards (with Add to Cart) and service cards (with details).
3 Add to Cart
From Browse Product cards or Product details page
If a product has variants (like size/color), you may need to select a variant on the product detail page.
4 Go to Cart + Checkout
`/cart` → `/checkout`
Checkout shows an order summary, address selection, and “Place order” (COD).

Your Customer Account

After logging in as a customer
1 Go to Customer dashboard
`/customer`
2 Manage your profile
`/customer/settings`
3 See your orders
`/customer/orders`
If you can’t see cart buttons: check you are logged in as customer. The cart is hidden for seller/admin accounts.

Product Seller Panel

Products, subscription limits, and settings
1 Open Product Seller overview
`/product-seller`
2 Products list
`/product-seller/products`
3 Add a new product
`/product-seller/products/new`
While creating, you set category, images, and variants (size/color) + stock.
4 Subscription Plans
`/product-seller/subscription`
Current plan affects restrictions like max number of products. (For testing, plan switching can be done without payment.)
5 Profile / Settings
`/product-seller/settings`

Service Seller Panel

Services, subscription limits, and settings
1 Open Service Seller overview
`/service-seller`
2 Services list
`/service-seller/services`
3 Add a new service
`/service-seller/services/new`
While creating, you set service type, base price/discount, images, and (if needed) weekly availability.
4 Subscription Plans
`/service-seller/subscription`
Current plan affects restrictions like max number of services.
5 Profile / Settings
`/service-seller/settings`

Admin Panel

Moderate marketplace
1 Open Admin overview
`/admin`
2 Manage orders
`/admin/orders`
3 Manage sellers
`/admin/sellers`
4 Manage subscriptions
`/admin/subscriptions`
5 Manage categories, services categories, banners, ads
`/admin/categories`, `/admin/service-categories`, `/admin/banners`, `/admin/admanagement`, `/admin/seller-ads`

Common Tips (Everyone)

Quick reminders
1 Make sure you are logged into the correct role
Cart + checkout are customer-only. Seller/admin accounts won’t see customer cart buttons.
2 For products, variants matter
If a product has variants (size/color), you must select a valid variant before “Add to Cart” from the product detail page.
3 For quick testing (subscription)
Subscription plan buttons are meant to be used for testing restrictions today. Payment flow can be added later.
Tip: if your browser blocks local HTML, copy the file to a safe folder and open it. This guide is static and does not require internet.
