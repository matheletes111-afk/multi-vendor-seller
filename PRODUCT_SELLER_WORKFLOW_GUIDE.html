3. How Pricing & GST Work at Checkout

Crucial Financial Info

Our marketplace handles pricing calculations transparently. When a customer adds items to their cart, the system calculates the price step-by-step:

$

Base minus Discount

If base price is 100 and discount is 10, the final listing price is 90.00. This is what is shown on the shop pages.

%

GST Flag Check

Each variant has a "Has GST" toggle. If checked, a 15% GST tax rate is calculated and added to this item at checkout.

+

Total Math Formula

Total Cost = (90.00 × Quantity).
If GST is checked: + (13.50 × Quantity).
Grand Total: 103.50

Important Note on Financials: Base prices and discounts are stored excluding GST. The 15% GST is computed dynamically on top of the discounted price at checkout, depending on whether the "Has GST" flag is active.

3. Bundling Multiple Variant Rows (e.g., 2 Variants)

To group multiple rows into a single product listing (with variant select buttons on the product page), keep the parent columns identical across those rows while changing the variant columns.

How to Upload 2 Variants Under One Product:

  1. Identify your product with a unique product_key (e.g. T-SHIRT-M1). Use this exact key for both rows.
  2. Enter matching parent values in both rows: category_id, product_name, description, and product_images.
  3. Differentiate the variants using variant_name (e.g. Blue / M and Red / L) and distinct sku values.
  4. Define variant-specific details such as price, stock, and filter configurations inside attributes_json.

Frequently Asked Questions (Troubleshooting)

What does the "A product slug conflict occurred" error mean during bulk import?
Every product has a unique URL slug based on its name. If you attempt to upload products with names that match existing listings in your inventory, a conflict might occur. Try renaming your product slightly or delete old items before re-uploading.
How do I write attributes_json in Excel or CSV without getting errors?
In Excel, simply write valid JSON like {"Color": "Blue", "Size": "L"}. In CSV, you must surround the entire JSON block in double quotes and escape internal quotes: "{""Color"":""Blue"",""Size"":""L""}". Keep it blank if you don't need custom filters.
Why is my upload showing "Import validation failed: subcategory_id does not belong to category"?
Ensure the subcategory_id you entered exists and is mapped to the main category_id in the system. If you aren't sure, download a fresh template after configuring categories in Step 1 of the upload modal — it will contain valid IDs in the dummy rows.
How many rows can I upload in a single file?
The maximum limit is 500 data rows per spreadsheet file. File size must be under 8 MB.
Can I edit existing products using Bulk Upload?
No, the bulk upload feature currently only supports importing new products. To edit existing products, please use the Edit form in the seller dashboard interface.
